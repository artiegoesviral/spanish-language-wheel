# Spanish Language Wheel

Interactive React + TypeScript application for practising Spanish verb conjugations through a draggable wheel interface.

## Features
- Drag-based interactive verb wheel
- Displays present and past conjugations
- Built with React, TypeScript and Vite
- Responsive UI structure
- Designed for language-learning practice

## Tech Stack
React, TypeScript, Vite, CSS

## Why I built it
This project demonstrates component-based frontend development, state management with React hooks, and user interaction handling.
